# vdl

Command line tool for downloading videos from youtube

## Getting Started

On a terminal:

```
// Download from link
vdl --url 'link'

// Download from file
// The file contains links or keywords
vdl --file 'playListFile.txt'
```

### Installing

```
npm install -g vdl
```
